# Project Dashboard Web App

ระบบ Dashboard สำหรับติดตามงานโครงการ โดยใช้ไฟล์ CSV เป็นข้อมูลนำเข้า  
รองรับการแสดงผลแบบ **Gantt Chart Timeline**, การอัปเดตสถานะงาน, การสรุปค่าใช้จ่าย, การแจ้งเตือนผ่าน Telegram และการบันทึกไฟล์ CSV ที่อัปเดตแล้ว

---

## 🚀 การติดตั้ง

1. ดาวน์โหลดชุดไฟล์ทั้งหมดจากโฟลเดอร์ `project-dashboard`
2. เปิดไฟล์ `index.html` ด้วย Browser (Chrome/Edge/Firefox)
3. ระบบพร้อมใช้งานทันที ไม่ต้องติดตั้ง Server

---

## 📂 โครงสร้างไฟล์

- `template.csv` → Template มาตรฐานสำหรับทุกปี
- `demo-2569.csv` → ตัวอย่างข้อมูลปี พ.ศ. 2569
- `demo-2570.csv` → ตัวอย่างข้อมูลปี พ.ศ. 2570
- `index.html` → Prototype Web App
- `README.md` → คู่มือการใช้งาน

---

## 📊 วิธีใช้งาน

1. กดปุ่ม **เลือกไฟล์ CSV** → เลือกไฟล์ข้อมูล (เช่น demo-2569.csv)
2. Dashboard จะแสดง:
   - Timeline ของทุกโครงการ
   - ค่าใช้จ่ายรวม และค่าใช้จ่ายแยกตามโครงการ
   - Filter เลือกดูเฉพาะโครงการ
3. คลิกที่งานใน Timeline → Toggle สถานะ (TRUE/FALSE)
4. ระบบจะส่ง **แจ้งเตือนผ่าน Telegram** เมื่อมีการอัปเดตสถานะ
5. กดปุ่ม **Export CSV** → บันทึกสถานะล่าสุดเป็นไฟล์ใหม่ (`updated_tasks.csv`)
